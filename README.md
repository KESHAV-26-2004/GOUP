# 📌 Overview

**GO UP** is a vertical 2D platformer game developed using **Python and the Pygame library**.

Players control a ninja frog character that must **jump across randomly positioned platforms**, avoid **animated fire traps**, and reach the **highest point in the game world**.

The project demonstrates core **game development concepts** such as:

* sprite animation
* collision detection
* gravity physics
* procedural platform generation
* menu systems and game states

The project was developed to explore **2D game development, physics simulation, and interactive gameplay systems using Pygame**.

# 🧠 Game Mechanics

The game implements several important programming concepts.

## Physics System

* Gravity-based movement
* Jump velocity and falling acceleration

## Collision Detection

The game uses **pygame masks** to detect collisions between:

* player and platforms
* player and traps

## Procedural Platform Generation

Platforms are generated using **randomized X positions while maintaining consistent vertical spacing**, creating a dynamic climbing experience.

## Sprite Animation

Character movement animations include:

* idle
* running
* jumping
* double jumping
* falling
* hit reaction
